# Personal Knowledge Base Graph

A small interactive knowledge graph viewer built with **Next.js + TypeScript + React Flow**.

Users can create topics (nodes) and connect them with labeled relationships (edges).
All graph data is persisted in **localStorage**.

## Features

- Interactive graph canvas
- Add and delete nodes
- Connect nodes with labeled edges
- Edit node title and note
- Delete edges
- Drag nodes to reposition
- Graph state persisted using localStorage

## Tech Stack

- Next.js
- TypeScript
- React Flow

## Running Locally

Clone the repository:

```
git clone https://github.com/Rohittalreja10/knowledge_graph.git
cd knowledge_graphv
```

Install dependencies:

```
npm install
```

Start development server:

```
npm run dev
```

Open:

```
http://localhost:3000
```

## Deployment

The project can be deployed using Vercel.
